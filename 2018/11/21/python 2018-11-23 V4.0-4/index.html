启程python

Posted on 2018-11-21 | In 纯小白的Python教程

看完这篇文章需要8分钟，实践完需要80分钟。

你会想象，学习一门编程语言的样子，是在那边敲代码。事实上，首先你会被其他一些东西卡住。比如说，来，先安装好Python。

安装python

第一次安装Python前，我听说ios系统自带python，我找了半天没有找到。小白第一次产生自我怀疑，但不会是最后一次。

  • 解决方案：python官网下载安装。
  • 后续思考：为什么当时python没有找到呢 ？
  • 最终结论：反正都装好了，不纠结。

运行第一个Python脚本

交互界面下，不是真的运行一个程序。正常步骤如下：

  • 用编辑器写一个程序
  • 保存成.py的脚本文件：比如abc.py
  • 在CLI中，cd命令打开脚本文本所在的文件夹，运行该脚本文件：python abc.py

即时反馈的交互式编程环境

关于编程环境的问题：对小白来说可以理解为IDE环境、终端环境、程序文件+终端执行，其实最后都是终端调用shell执行。—— 大妈

IDE（Integrated Development Environment）,集成开发环境,辅助程序开发人员开发软件的应用软件，在开发工具内部就可以辅助编写源代码文本、并编译打包成为可用的程序，有些甚至可以设计图形接口。
D: development、design、debugging

终端交互界面

在终端中输入python,可以显示当前python版本，并且进入python 交互界面的。

我一开始不知道有这个交互界面，学习效率很低。后来知道了很惊喜，原来这么容易就可以获得这样的交互界面，输出代码，马上可以看到结果，反馈即时，学习体验很友好，效率提高很多。

IDLE交互界面

下载python自带的交互界面。和命令行界面相比，更加人性化，不同的内容会标出不同的颜色。

选什么交互界面？

大妈推荐

  • IDLE和terminal都没有运行
  • 推荐开发环境推荐使用sublime编辑器（理由：颜色识别，自动函数等）
    Ipython= notebook+jupyter；notebook优点是，网页上同时展示，环境，思路，代码同时展示；限制是，比较难装且与标准程序开发环境有所区别
    ipython notebook，一个基于web的ipython shell！notebook使用起来和ipython差不多，不过修改起来方便得多，并且可以建多个notebook，所以使用起来各种方便

我没有用过sublime，学习Python只在terminal的交互环境中；后来我学习深度学习的时候，是在Idocker环境下的jupyternotebook, 体验很好。

交互界面的及时反馈，对于Python这样一门编程语言的学习很重要，展开来说，对于其他学科的学习如此，对于个人能力和表现提升也是如此。

##

热身练习《Learn Python the Hard Way》

这本书是比较公认的入门教程，总共是52个练习。它强调动手，在练习中学习。类似鼓励先下水，划水扑腾中学会游泳的教学法。不过动作切分得很小，新手不会被一口呛死。热身可以从这本练习开始，通过简单的代码运行成功，获得正面的反馈。然后从输入代码和运行结果的预测和验证中，感知python语言的规则。

Python2和Python3

通常大家的经验是，软件会不断更新，一般来说，选择最新版本不会有啥问题，新版本一般就是增加新功能，bug修复，体验优化，不同版本可以兼容。

但是python2和python3不兼容，也就是说，用其中一种语言写的代码，用另一个版本是打不开的。感受一下，有点像这种情况：同样在吴语区，上海话和苏州话也不一样，彼此不完全能听懂。

《learn python the hard way》这本书是基于Python2教的，并且并没有告诉Python2和Python3的区别。

python2和python3如何切换？

具体场景的问题描述：

  • 怎么知道，当前运行代码的是python2还是python3呢？
  • 如何在电脑中切换这两种语言呢？
  • 遇到的困难不知道是p2还是p3在运行,怎么切换python3运行？

曾经被一个方案绕进去很久，折腾了一两个小时也没有成功。最后解决方案其实很简单：

  • 命令行python，改成python3。
  • 安装各种扩展，记得要安装在不同的版本下。安装在python2下面，python3是不能用的，反过来也是一样。

python2和python3的语法区别

两者之间差别有好多，用《learn python the hard way》中第1个到第3个练习改写来说明。

1. print 变成了函数，print后面的文本要加上（）

用exercise2示例说明：
python2

python3

2. 因为支持了Unicode (utf-8)，所以可以识别中文

用exercise2示例说明：
python2

python3

3. 输入:raw_input变成input

help()和dir()获取帮助信息

help() 交互模式下，用help向python请求帮助
help() 具体的模块帮助信息
dir() 内置dir()函数检查模块（以及其对象）的内容
quit() 回到交互状态
