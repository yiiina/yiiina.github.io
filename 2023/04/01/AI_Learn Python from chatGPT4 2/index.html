2. 计算 1 到 10 的和

这段代码的作用是计算从1到10的所有整数的和，并将结果打印出来。

变量sum被初始化为0，然后通过一个for循环迭代1到10的所有整数。在每次循环中，当前整数i被加到sum变量中。循环结束后，变量sum包含从1到10的所有整数的和。最后，使用print()函数打印出sum的值。

3. 判断一个数是奇数还是偶数

这段代码的意思是让用户输入一个整数，然后判断它是偶数还是奇数。如果它是偶数，则输出它是偶数；否则输出它是奇数。这是一个基本的条件语句，其中if语句用于检查一个条件，如果该条件为真，则执行相应的代码块；否则执行else语句块中的代码。在这个例子中，我们使用了取模运算符（%）来检查一个数是否是偶数，因为如果一个数是偶数，它对2求余的结果将是0。

4. 将一个字符串反转

Enter a string: 是的
Reverse string is: 的是

这段代码要求用户输入一个字符串，然后将字符串反转并输出。

代码的第一行使用 input 函数要求用户输入一个字符串，并将其存储在变量 str 中。

第二行使用字符串的切片操作 [::-1] 反转字符串。这个操作将从字符串的末尾开始，步长为 -1（即从右到左），一直到字符串的开头。这样就得到了一个反转后的字符串，保存在变量 reverse_str 中。

第三行使用 print 函数输出反转后的字符串。输出信息中包含字符串 “Reverse string is: “，后面跟着反转后的字符串 reverse_str。
